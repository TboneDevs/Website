# TNNR CPM1 Webtool

Inject unlocks into **existing** CPM1 accounts (Malik logic).

## Deploy (Railway)
1. Push this folder to GitHub
2. New Railway project → Deploy from repo
3. Done

## Local
```bash
pip install -r requirements.txt
python app.py
```

## Features
- Full Unlock one-click
- Custom money / coins / race wins
- King Rank, Levels, W16, No Damage, Fuel, Smoke, Horns, Animations, Wheels, Houses, Equipment (Headless)
- Free Accounts button → www.TNNR.lol
